# NGS BLAST Workshop Assignment

This repository contains my submission for the Online NGS Workshop activity titled
"Identification of Unknown Biological Sequences Using NCBI BLAST".

## Contents
- Report: Detailed BLAST analysis and interpretation of three unknown sequences
- Screenshots: BLAST result screenshots for all analyzed sequences

## Tools Used
- NCBI BLAST (BLASTn and BLASTp)

## Summary
Three unknown biological sequences were analyzed to determine their sequence type,
organism of origin, and biological function using sequence similarity analysis.
